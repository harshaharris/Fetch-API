# Fetch-API
Description:
      Task: Fetch and display user data using JavaScript Fetch API.
Objective: Retrieve user details (username, phone, company, website) from https://jsonplaceholder.typicode.com/users.
Tools Used: HTML, CSS, JavaScript (Fetch API), Chrome Browser, VS Code.
Features: Displays user cards, handles fetch errors, includes reload functionality.
Outcome: Successfully demonstrated dynamic data fetching and DOM manipulation with error handling.
